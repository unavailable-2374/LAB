Fastp

A tool designed to provide fast all-in-one preprocessing for FastQ files. This tool is developed in C++ with multithreading supported to afford high performance.

samtools

Tools (written in C using htslib) for manipulating next-generation sequencing data

htslib:C library for high-throughput sequencing data formats

GATK4

GATK4 aims to bring together well-established tools from the GATK and Picard codebases under a streamlined framework.

BWA

BWA is a software package for mapping DNA sequences against a large reference genome, such as the human genome. It consists of three algorithms: BWA-backtrack, BWA-SW and BWA-MEM.

FastQC

FastQC is a program designed to spot potential problems in high througput sequencing datasets. It runs a set of analyses on one or more raw sequence files in fastq or bam format and produces a report which summarises the results.

FastQC

tabix can index files in common formats in NGS analysis to speed up the access speed. It not only supports VCF files, but also supports BED, GFF, SAM and other formats.
